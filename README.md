# Credit Card Financial Dashboard

## Overview
This project is an interactive Power BI dashboard built to analyze Credit Card Customer and Transaction data.

## Features
- Revenue Analysis
- Interest Earned Analysis
- Customer Segmentation
- Revenue by Age Group
- Revenue by Education
- Revenue by Income Group
- Transaction Analysis
- Expenditure Type Analysis
- Card Category Analysis

## Tools Used
- Power BI
- SQL
- CSV Dataset

## Files
- credit_card_Customer_report.pdf
- credit_card_Transaction_report.pdf
- credit_card2.csv
- customer.csv
- SQL Query - Financial Dashboard Data.sql
